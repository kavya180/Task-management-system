A full-stack task management application built with React, Node.js, Express, and MongoDB.

## Features
User authentication with register and login functionality. Complete CRUD operations for tasks including create, read, update, and delete. Tasks can be organized by categories (Work, Personal, Shopping, Health, Education, Other), assigned priority levels (Low, Medium, High), and tracked through statuses (Todo, In Progress, Completed). The application includes due date management, task filtering by status, a statistics dashboard, and a responsive design that works on all devices.
## Tech Stack

**Frontend:** React 18, React Router DOM, Axios, React Icons, Vite

**Backend:** Node.js, Express.js, MongoDB with Mongoose, JWT Authentication, bcryptjs
